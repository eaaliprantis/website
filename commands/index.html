[] = required, () = optional
Command Parameters Description Example
$changelog Gives you my change log. $changelog
$ping Just to check to see if TypicalBot is working. $ping
$commands Gives you a list of commands $commands
$cmds Gives you a list of commands $cmds
$help Get some help with TypicalBot! $help
$shoot (@USER) Digitally shoot a user by mentioning them. $shoot @Hackzzila
$stab (@USER) Digitally stab a user by mentioning them. $stab @Hackzzila
$cat Sends you a cute cat. $cat
$dog Sends you a cute dog. $dog
$penguin Sends you a random penguin. $penguin
$image [QUERY] Searches the image [QUERY] and sends it in chat. $image Funny cats
$joke (yomomma/chucknorris) Sends you either a yomomma joke, Chuck Norris joke, or random joke (if there is no argument). $joke chucknorris
$troll [MIN]
[MAX]		 Rolls a number between [MIN] and [MAX]. $troll 1 6
$thisserver Shows your statistics for the server. $thisserver
$userinfo (@USER) Gets the information for a specific user mentioned. $userinfo @Hackzzila
$discriminator (####) Gets all of the users I have seen with the discriminator you supply. $discriminator 7109
$8ball [QUESTION] Use the magic 8ball.
$bans Sends you the list of people banned from the server you say it in. (Only works if bot has ability to see bans, Ban Member permission.) $bans
$bots Detects a list of other bots in the server. $bots
$invite Sends you his OAuth invite URL. $invite
$stats Shows the uptime, amount of servers I am in, and amount of users total of all of the servers. $stats
[] = required, () = optional
Command Parameters Description Example
$say [MESSAGE] Makes TypicalBot say whatever you put in for [message]. $say I love Hackzzila
$csay [#CHANNEL]
[MESSAGE]		 Makes TypicalBot say whatever you put in for [message] in the specificed channel. $csay #general Hackzzila is the best person on Earth.
$prune [#] Prunes the last [#] messages. $prune 50
$kick [@USER] Kicks [@USER]. $kick @TypicalModerator
$softban [@USER] Soft bans [@USER]. (Ban + Remove Past Messages + Uban, similar to kick) $softban @TypicalModerator
$ban [@USER] Bans [@USER] $ban @TypicalModerator
$announce [set/off] Sets up a join/leave/ban announcement in the specificed channel. $announce set
$joinmessage [set/off]
(message)		 Sets up a join message to send to a user when they join your server. $joinmessage set Welcome human!
$antilink [on/off] Turn on or off an anti-link setting. $antilink on
$antiinvite [on/off] Turn on or off an anti-invite setting. $antiinvite off
$imageblock [on/off] Turn on or off an image block (Disallows people from searching bad things with the image command.) $imageblock on
$customprefix [set/off]
(prefix)		 Sets a custom prefix for your server to whatever you put for (prefix). $customprefix set --
$cprefix [set/off]
(prefix)		 Shortcut of $customprefix [set/off] (prefix) $cprefix off
$rolecolor [#HEX]
[ROLE NAME]		 Changes a role's color to the [HEX] color provided. $rolecolor #ffffff Admins
$role [give/take]
[@user]
role_name]		 Give/take a role from a player. (Use <&> instead of spaces in the role's name.) $role give @Hackzzila Admins
$leave Makes TypicalBot leave a server. $leave
